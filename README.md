# GitHub Star Tracker

⚡ 极客风格 GitHub 热门项目趋势追踪器

## 功能特性

- 🔥 实时追踪 GitHub 热门项目（每6小时自动更新）
- 📊 Star 增长趋势可视化图表
- 🏷️ 标签筛选功能
- 🔍 项目搜索功能
- 📱 响应式设计，适配各种设备

## 技术栈

- Vue 3
- Vite
- Chart.js + vue-chartjs
- GitHub API

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 部署

项目已配置 Vercel 部署，直接推送后自动部署。
